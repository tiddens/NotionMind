# Getting Started

## Keyboard Shortcuts

### Navigation

#### Arrow keys to move between nodes

#### Escape to return to root

### Editing

#### Tab to add a child node

#### Enter to add a sibling node

#### Delete to remove a node

#### F2 or just start typing to edit

### Organization

#### Space to fold/unfold

#### Ctrl+Up/Down to move nodes

#### Ctrl+Left/Right to move sides

#### Auto save

## Tips

### Keep it Simple

#### One idea per node

#### Use short phrases

### Stay Organized

#### Group related ideas

#### Use folding to focus
